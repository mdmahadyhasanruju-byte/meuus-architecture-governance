## Purpose

This Draft document will define how public-facing meUus institutional expression must be governed, constrained, reviewed, labeled, and corrected.

It is a Draft stub only. It is not Working, not Ready for Review, not Frozen, and not canonical.

## Scope

This document is intended to later clarify:

- what `meuus.org` may publicly express.
- what public institutional truth means.
- what public pages may claim.
- what public pages must not claim.
- how status labels should appear publicly.
- how evidence supports public expression.
- how public wording must obey `Claim <= Evidence`.
- how public expression differs from canonical knowledge.
- how public expression differs from runtime reflection.
- how public expression differs from governance authority.
- what must be escalated before publication.
- how public mistakes or outdated claims should be corrected.
- how Amanah applies to public institutional communication.

This Draft stub does not yet define those rules.
