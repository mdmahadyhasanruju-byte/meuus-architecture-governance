# Public-Expression Review Plan

## Governance Boundary

This plan does not approve publication.

This plan does not make the manuscript canonical.

This plan does not amend Frozen governance.

This plan does not create website copy.

This plan only defines the public-expression review route.

Public expression must obey Documents 05, 06, and 07.

Public expression cannot create authority.

Book voice cannot override governance authority.

Source material cannot become public truth without evidence, provenance, and review.

## Public-Expression Principles

- Claim <= Evidence.
- Public language must not exceed actual status.
- Future systems must be labeled future/planned.
- Prototypes must be labeled prototype/demo/bounded.
- Vision must be separated from live capacity.
- Founder-origin narrative must not become proof.
- Personal pain must not become marketing.
- Religious language must remain reflection/learning, not authority.
- Legal language must remain literacy, not advice/representation.
- Therapy/recovery language must remain educational/reflection, not treatment.
- AI/DLAS language must remain future/research/reflection unless validated.
- Public trust/safety claims require operational evidence.
- Status labels must remain visible.
- Private material stays private unless reviewed and abstracted.
